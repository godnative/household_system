# 开发规则说明

## 1. 开发模式启动命令
在开发模式下，启动命令需要带上`--debug`参数，这样可以跳过用户登录，直接调试主界面代码。

```bash
python -m src.app --debug
```

## 2. 测试步骤
当启动应用进行测试时，由于是GUI桌面程序，需要按照以下步骤进行测试：

### 测试对话框功能
1. **测试村庄管理**：
   - 点击"村管理"标签
   - 点击"添加村庄"按钮，测试添加对话框
   - 点击"编辑"按钮，测试编辑对话框

2. **测试家庭管理**：
   - 点击"家庭管理"标签
   - 点击"添加家庭"按钮，测试添加对话框
   - 点击"编辑"按钮，测试编辑对话框

3. **测试成员管理**：
   - 点击"成员管理"标签
   - 点击"添加成员"按钮，测试添加对话框
   - 点击"编辑"按钮，测试编辑对话框

## 3. 错误处理
在测试过程中如果遇到问题，会有错误代码报错，需要分析错误原因并修复。如果测试成功，请点击正常退出按钮，应用应该会无错误代码退出。
